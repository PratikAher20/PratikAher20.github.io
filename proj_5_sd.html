GRACE which stands for GMC, Reprogramming and Communication Experiment is the sequel of PiLOT payload. Main objectives of this payload is to demonstrate the first ever In Orbit Reprogramming operation on a Cubesat sized payload, space qualify the Gieger-Muller-Counter payload which counts the number of radiations in space, and space qualify the CubeSat UHF Board for receiving the commands sent from the IIST ground station. This payload is to be launched on POEM-4 platform of PSLV-C60 mission.
  • Work Done
    • Contributed to the development of CubeSat UHF board device drivers and interfaced with the OBC.
    • Contributed to the development of GMC payload device drivers and interfaced with the OBC.
    • Improved the device drivers interfacing our payload with the POEM package for data transmission.
    • Added device drivers to receive commands sent from the POEM packages for our payload.
    • Designed and implemented a complete flight software architecture for the mission.
    • Added the reprogramming functionality to the software.
